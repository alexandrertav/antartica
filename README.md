# WebMapa Antártica - Ilha Rei George

Mapa interativo com fotografias georreferenciadas e dados geomorfológicos da Ilha Rei George, Antártica.

## Tecnologias

- Next.js 15
- React 19
- Leaflet
- Tailwind CSS

## Desenvolvimento

```bash
npm install
npm run dev
```

Acesse: http://localhost:3000

## Produção

```bash
npm run build
npm start
```

## Estrutura

```
app/           → Páginas (Next.js App Router)
src/components → Componentes React
src/hooks      → Hooks customizados
public/        → Assets estáticos
```
